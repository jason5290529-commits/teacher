製作心得

這次期末專題讓我從零開始練習製作一個完整網站。 透過 HTML 建立網頁內容，使用 CSS 設計版面， 並加入 JavaScript 互動效果，讓網站不只是靜態頁面。

雖然一開始對網頁設計不熟悉，但透過一步一步完成每個頁面， 我更了解網站製作的基本流程，也對前端設計有了初步認識。
