選擇這位老師的原因

我會選擇陳建彰教授，是因為他是我大一的班級指導老師。 對剛進入大學的我來說，班導是一位很重要的引導者， 讓我能更快認識大學生活與資訊工程學系。

另外，在入學面試時，陳建彰教授也讓我留下深刻印象。 因此這次期末專題，我希望透過網頁設計的方式， 介紹這位對我大學初期具有重要意義的老師。

我的學習收穫

透過製作這個網站，我不只更認識老師的學經歷與研究專長， 也練習了 HTML、CSS 與 JavaScript 的基本應用。

這次專題讓我理解到，一個網站不只是把文字放上去， 還需要考慮畫面排版、使用者操作、手機瀏覽效果以及互動功能。

HTML

負責建立網頁架構，例如標題、段落、導覽列與區塊。

CSS

負責設計網頁外觀，例如顏色、排版、間距與卡片效果。

JavaScript

負責增加互動功能，例如按鈕提示、滑鼠移入效果與輸入顯示。
